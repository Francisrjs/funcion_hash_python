# Función hash simple

Pequeño script en Python 3 que genera el hash hexadecimal de un texto utilizando el algoritmo que elijas (por defecto `sha1`).

## Requisitos
- Python 3.11 o superior (probado en Windows)

## Uso
1. Clona el repositorio:
   ```bash
   git clone https://github.com/Francisrjs/funcion_hash_python.git
   cd funcion_hash_python
   ```
2. Ejecuta el script e ingresa tus datos (ejemplo incluido en el archivo):
   ```bash
   python funcion_hash_simple.py
   ```

Modifica la llamada final de `mi_hash_simple` con el texto o algoritmo que prefieras (`md5`, `sha256`, etc.).
